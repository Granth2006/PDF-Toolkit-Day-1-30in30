PDF Toolkit

A powerful, browser-based PDF editor with 12 tools — merge, split, compress, convert, watermark, sign and more. No uploads. No servers. Everything runs locally.

🚀 Live Demo ⭐ GitHub

⚙️ Features

🔗
Merge PDFs
Combine multiple PDF files into one. Drag to reorder before merging.
✂️
Split PDF
Extract a page range, split each page, or divide into N equal parts.
🗜️
Compress PDF
Reduce file size with adjustable image quality and optimization level.
📝
PDF to Text
Extract all readable text from a PDF. Copy or download as .txt.
🖼️
Image to PDF
Convert JPG / PNG / WebP images into a PDF. Supports multi-image, A4, Letter, or fit-to-image.
🔀
Reorder Pages
Visual drag-and-drop interface to rearrange PDF pages effortlessly.
🗑️
Delete Pages
Select specific pages to remove from a PDF using a thumbnail grid.
💧
Add Watermark
Stamp diagonal text watermarks with custom color, size & opacity.
🔢
Page Numbers
Auto-stamp page numbers in 6 positions with multiple format options.
🔄
Rotate Pages
Rotate individual pages or bulk-rotate all pages by 90° / 180°.
🔒
Protect PDF
Add open/owner password protection. Encrypted client-side.
✍️
Sign PDF
Draw your signature on a canvas and embed it into any page.
🔐
100% Private — Your Files Never Leave Your Device
All PDF operations are performed entirely in the browser using JavaScript. No file is ever uploaded to any server. Everything stays local on your machine.
