# 📄 PDF Toolkit

### A powerful, browser-based PDF editor with 12 tools.<br/>No uploads. No servers. Everything runs locally in your browser.

<br/>

[![🚀 Live Demo](https://img.shields.io/badge/🚀%20Live%20Demo-pdf--toolkit--30in30.vercel.app-7c3aed?style=for-the-badge)](https://pdf-toolkit-30in30.vercel.app/)
&nbsp;&nbsp;
[![GitHub](https://img.shields.io/badge/⭐%20GitHub-Granth2006-24292e?style=for-the-badge&logo=github)](https://github.com/Granth2006)

## ⚙️ Features

<table>
  <tr>
    <td width="50%">
      <h3>🔗 Merge PDFs</h3>
      Combine multiple PDF files into one document. Drag to reorder before merging.
    </td>
    <td width="50%">
      <h3>✂️ Split PDF</h3>
      Extract a page range, split each page individually, or divide into N equal parts.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🗜️ Compress PDF</h3>
      Reduce file size with adjustable image quality (10–100%) and optimization level.
    </td>
    <td>
      <h3>📝 PDF to Text</h3>
      Extract all readable text from a PDF. Copy to clipboard or download as <code>.txt</code>.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🖼️ Image to PDF</h3>
      Convert JPG / PNG / WebP images into a PDF. Supports A4, Letter, or fit-to-image.
    </td>
    <td>
      <h3>🔀 Reorder Pages</h3>
      Drag-and-drop visual interface to rearrange pages in any order effortlessly.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🗑️ Delete Pages</h3>
      Select specific pages via a thumbnail grid and remove them from the PDF.
    </td>
    <td>
      <h3>💧 Add Watermark</h3>
      Stamp diagonal text watermarks with custom color, font size & opacity.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🔢 Page Numbers</h3>
      Auto-stamp page numbers in 6 positions with multiple numbering formats.
    </td>
    <td>
      <h3>🔄 Rotate Pages</h3>
      Rotate individual pages or bulk-rotate all by 90° left / right / 180°.
    </td>
  </tr>
  <tr>
    <td>
      <h3>🔒 Protect PDF</h3>
      Add open & owner password protection, encrypted entirely client-side.
    </td>
    <td>
      <h3>✍️ Sign PDF</h3>
      Draw your signature on a canvas and embed it onto any page of a PDF.
    </td>
  </tr>
</table>

---

## 🔐 Privacy First

> **Your files never leave your device.**
> All PDF operations run entirely in the browser via JavaScript.
> No file is ever uploaded to any server. Everything stays local.

## 🧰 Tech Stack

| Technology | Purpose |

| ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) | Structure & markup |
